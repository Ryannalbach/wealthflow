# WealthFlow

A modern budget calculator app built with React and Tailwind CSS.

## Features
- Plan your monthly budget
- Track debts, expenses, savings, and investments
- Visualize your financial distribution with charts
- Auto-save data to browser localStorage

## Live Demo
Visit: https://ryannalbach.github.io/wealthflow/

## Local Development
```bash
npm install
npm run dev
```

## Build for Production
```bash
npm run build
```
